# Front - CSS1

## 1. CSS

### CSS (Cascading Style Sheets)

문서 내용과 상관없이 스타일을 처리하는 기술

수시로 변경되는 웹 화면의 디자인 요소를 HTML에서 분리

**장점**

문서와 디자인의 분리로 소스의 관리와 수정이 용이함

PC, 모바일 등 다양한 기기 별 디자인 적용 용이

### 기본 문법

CSS는 선택자와 선언부로 구성

선택자; 정의할 스타일을 적용할 대상

선언부: 선택자에 적용할 스타일로 { } 안에 작성하며 여러 개일 경우 ; 으로 구분

- 속성: 값

**스타일 적용 방법**

- 인라인 스타일

개별 태그들에 style=”property:value;” 형태로 지정

선택자 사용하지 않으며 재사용 불가

다른 스타일에 비해 강력한 우선 순위를 가짐

- 내부 스타일

<head>의 <style>에 집중해서 작성

해당 html 페이지 내에서는 선택자 기준으로 재사용 가능

페이지에 특화된 내용 작성

![image.png](4ce2d1f4-e104-4d34-bce7-546dbfdf2d19.png)

- 외부 스타일 시트 활용

외부에 별도의 스타일 시트 파일(.css)를 만들고 <link> 태그로 연결

모든 페이지에서 재사용 가능

### 적용 순서

Cascading: 여러 스타일 규칙이 적용될 때 우선순위를 결정하는 방식

즉 style은 계층을 갖는다.

**계층 결정 방식**

중요도 > 명시도 > 소스 순서 > 상속

**중요도**

style sheet는 **중요도에 따라 3가지 단계의 origin으로 구성** - user agent, user, author

- user agent: 웹 브라우저가 미리 설정해 놓은 스타일로 가장 낮은 우선순위
    - “이 브라우저의 기준은 이렇습니다!”
- user: 사용자가 자신의 브라우저 설정이나 사용자 스타일 시트에 적용한 스타일
    - “나는 이렇게 보고 싶어” - 거의 사용 X
- author: 웹페이지를 작성한 개발자가 정의한 스타일
    - “우리 사이트는 이렇게 보이고 싶어!”

!important는 중요한 속성에 지정함

**명시도**

만약 중요도가 같다면 다음으로 명시도 확인

**선택자를 이용해서 우선순위를 정하기 위한 값**

**“얼마나 한정지을 수 있는가?”**

만약 명시도까지 같다면 로드된 우선순위에 의해 결정

**상속**

부모 요소에 지정한 스타일이 자식 요소에도 적용

## 2. Selector

### 선택자?

CSS 규칙을 적용할 대상을 선택하기 위한 표현식

명시도는 **선택자를 이용해서 우선순위를 정하기 위한 값**

**얼마나 한정지을 수 있는가?**

X-Y-Z의 3가지 숫자로 명시도를 매기며 마치 금은동과 유사

### CSS 선택자

**기본 선택자**

전체 선택자(*): 문서 내 모든 요소를 선택. 주로 user agent의 css를 reset하는 용도

태그 선택자: 지정된 태그명을 가진 요소를 선택

클래스 선택자(.): 특정 클래스 속성값을 가진 요소를 선택

ID 선택자(#): 특정 id 속성값을 가진 요소를 선택. ID는 문서 내에서 유일해야 함

**복합 선택자**

자식 선택자(>)

자손 선택자(공백)

인접 형제 선택자(+)

일반 형제 선택자(~)

**속성 선택자**

[attr]: 특정 속성을 가진 요소를 선택

[attr=val]: 특정 속성의 값이 정확히 일치하는 요소를 선택

**가상 클래스 선택자**

실제로 class 형태로 존재하지는 않지만 상황에 따라 적용되는 class를 지정하기 위한 것

개발자가 태그에 class를 지정할 필요는 없으며 class 이름 앞에 : 추가

## 3. box model

텍스트, 이미지 등의 모든 콘텐츠를 사각의 박스 형태로 관리하는 모델

HTML의 모든 요소는 사각형의 박스 모델

box는 content, padding, border, margin으로 구성

**padding 영역**

content와 border 사이의 여백

**border 영역**

padding을 밖에서 감싸고 있는 테두리

**margin 영역**

border 외부에서 다른 요소와의 거리
